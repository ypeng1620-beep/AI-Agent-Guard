# AI Agent Guard 快速接入教程

## 🚀 5分钟快速上手

### 步骤1：安装

```bash
cd D:\ai-agent-guard\integrator
npm install
```

### 步骤2：引入集成器

```javascript
const Integrator = require('./integrator/index.js');

// 创建实例
const guard = new Integrator({
  evalModel: 'local',      // 评估模式：local/doubao/zhipu/kimi
  defaultScenario: 'general'  // 场景：general/tts/code
});
```

### 步骤3：开始使用

```javascript
// 评估输出
const evalResult = await guard.evaluate(output, task);

// 检查约束
const checkResult = guard.checkConstraints(output);

// 记录收敛
const convResult = guard.recordAndCheck(score, text);

// 保存记忆
guard.saveMemory(taskId, result, 'general');
```

## 📋 场景化配置示例

### 场景1：通用对话助手

```javascript
const guard = new Integrator({
  evalModel: 'doubao',
  defaultScenario: 'general',
  config: {
    forbiddenWords: ['作弊', '外挂', '暴力'],
    maxLength: 5000,
    maxTurns: 20,
    scoreThreshold: 70,
    similarityThreshold: 85
  }
});
```

### 场景2：TTS语音合成

```javascript
const guard = new Integrator({
  evalModel: 'zhipu',
  defaultScenario: 'tts',
  config: {
    forbiddenWords: [],
    maxLength: 2000,
    maxTurns: 5,
    scoreThreshold: 75,
    similarityThreshold: 90,
    // TTS专用规则
    ttsRules: {
      prohibitSSML: true,
      maxDuration: 60
    }
  }
});
```

### 场景3：代码助手

```javascript
const guard = new Integrator({
  evalModel: 'kimi',
  defaultScenario: 'code',
  config: {
    forbiddenWords: ['eval', 'exec', 'os.system'],
    maxLength: 10000,
    maxTurns: 30,
    scoreThreshold: 80,
    similarityThreshold: 75,
    // 代码专用规则
    codeRules: {
      securityCheck: true,
      syntaxCheck: true
    }
  }
});
```

## ⚡ 常见问题

### Q: 评估分数偏低怎么办？
A: 降低 scoreThreshold 或启用语义增强层提升检测精度。

### Q: 正常内容被拦截？
A: 检查 forbiddenWords 列表，可能包含误伤词汇。

### Q: 收敛检测过于敏感？
A: 调高 scoreThreshold 或 similarityThreshold。

### Q: 浏览器模式超时？
A: 检查网络连接，或切换到 local 模式。
