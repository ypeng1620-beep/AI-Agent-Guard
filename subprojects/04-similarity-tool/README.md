# 内容相似度计算工具

基于 TF-IDF + 余弦相似度算法的文本相似度计算工具，支持普通文本和 TTS 脚本场景。

## 功能特性

- **余弦相似度算法**：将文本转为 TF-IDF 向量，计算余弦相似度，返回 0-100% 的百分比
- **中文优化**：内置中文分词（单字/双字/三字词抽取），针对中文语义优化
- **TTS 场景专项**：自动过滤时间戳 `[00:00:00]` 和格式符号 `{}` `[]` `()`
- **无外部依赖**：纯原生 JavaScript 实现，可直接运行
- **批量计算**：支持批量计算多个候选文本与参考文本的相似度

## 安装

无需安装，将 `similarity.js` 拷贝到项目中使用即可。

## 使用方法

### 基础引用

```javascript
const SimilarityTool = require('./similarity');
const tool = new SimilarityTool();
```

### 计算两段文本的相似度

```javascript
const score = tool.calculate('今天天气很好', '今日气候不错');
console.log(score); // 例如: 85.23
```

### TTS 场景专用计算

```javascript
const tts1 = '[00:00:00] 你好世界[00:01:30] 欢迎收听';
const tts2 = '[00:00:00] 你好世界[00:02:00] 欢迎收听';

const score = tool.calculateForTTS(tts1, tts2);
console.log(score); // 例如: 95.0
```

### 批量计算

```javascript
const reference = '今天天气很好';
const candidates = [
  '今天天气不错',
  '明天要下雨',
  '今日气候宜人',
  '晚上有电影'
];

const results = tool.batchCalculate(reference, candidates);
// 返回按相似度降序排列的结果
// [
//   { text: '今天天气不错', score: 92.5 },
//   { text: '今日气候宜人', score: 88.3 },
//   { text: '晚上有电影', score: 12.1 },
//   { text: '明天要下雨', score: 8.7 }
// ]
```

## 验收标准

| 测试场景 | 期望相似度 |
|---------|-----------|
| 相同文本 | 100% |
| 仅改标点/近义词 | ≥ 90% |
| 实质修改 | ≤ 70% |

## TTS 脚本清洗规则

`calculateForTTS()` 方法会自动处理以下格式：

| 格式 | 示例 | 处理方式 |
|------|------|---------|
| 时间戳 | `[00:00:00]` `[00:01:30]` | 移除 |
| 花括号 | `{语速:1.5}` | 移除 |
| 方括号 | `[高兴]` `[悲伤]` | 移除 |
| 圆括号 | `(很棒)` | 移除 |

## 算法说明

### TF-IDF 向量化

1. **分词**：中文采用 n-gram 方式提取 1-3 字词，英文提取单词
2. **TF 计算**：词频归一化（词频 / 总词数）
3. **IDF 计算**：逆文档频率 log((文档数+1)/(词出现文档数+1)) + 1
4. **TF-IDF**：TF × IDF

### 余弦相似度

```
cosine(a,b) = (a·b) / (||a|| × ||b||)
```

结果乘以 100 转换为百分比。

## 应用场景

- **AI 内容审核**：检测生成的 TTS 脚本是否与样本库中的内容高度相似
- **去重检测**：识别相似/重复的文本内容
- **抄袭检测**：文本相似度分析
- **TTS 质量控制**：确保生成的语音脚本与原版脚本的语义一致性
